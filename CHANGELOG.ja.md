# 変更履歴

このプロジェクトのすべての重要な変更はこのファイルに記録されます。

フォーマットは [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に基づいており、
[Semantic Versioning](https://semver.org/lang/ja/spec/v2.0.0.html) に準拠しています。

## [Unreleased]

## [0.7.0] - 2025-01-31

### Added
- マリオテーマ（SFC/任天堂スタイルUI）を追加
- マリオテーマをGTDモードとカンバンモードの両方に対応
- ドラゴンクエストスタイルのステータスヘッダー（職業、レベル、HP、MP）を追加
- ドラゴンクエストテーマにタスク詳細表示を追加
- DQ/マリオテーマのフッターにi18n対応を追加

### Fixed
- DQテーマでのタスク詳細からのプロジェクトリンクを修正
- DQ/マリオテーマでのコンテキスト設定（cキー）サポートを修正
- フッターのヘルプテキストでc=contextが欠けていた問題を修正

## [0.6.0] - 2025-01-31

### Added
- ドラゴンクエストRPGスタイルUIをTitledBoxコンポーネントで実装
- GTDモードとカンバンモードにDQスタイルの2カラムレイアウトを追加
- ドラゴンクエストスタイルのスプラッシュ画面（表示時間設定可能）を追加
- スプラッシュサブコマンド（`floq config splash`）を追加
- DQスタイルコンポーネントに検索機能を追加
- HelpModalの全タブにスクロールサポートを追加
- READMEにデモGIF（英語版と日本語版）を追加

### Fixed
- DQスタイルコンポーネントでのHelpModal入力処理を修正
- Modernテーマのtextmuted視認性を改善

## [0.5.0] - 2025-01-30

### Added
- カンバンモードの検索結果にカンバンラベルを表示

### Changed
- 完了タスクをデフォルトで直近7日間のみ表示するよう変更
- READMEを最新機能に合わせて更新

## [0.4.0] - 2025-01-30

### Added
- タスクフィルタリング用のコンテキスト機能（@work、@homeなど）を追加
- コンテキストCLIコマンド（list、add、remove）を追加
- addコマンドに`--context` / `-c`オプションを追加
- TUIでコンテキストによるフィルタリング（`@`キー）を追加
- TUIでタスクにコンテキストを設定（`c`キー）を追加
- タスク項目にコンテキストバッジ表示を追加
- タスク詳細ビューにコンテキスト表示を追加
- TUIから直接新しいコンテキストを追加可能に

## [0.3.1] - 2025-01-29

### Fixed
- リポジトリURLをpolidog/floqに修正

## [0.3.0] - 2025-01-29

### Added
- Vimライクなアンドゥ/リドゥ機能を追加

### Fixed
- ヘルプモーダルの入力処理とスプラッシュ画面のテーマカラーを修正

## [0.2.3] - 2025-01-29

### Improved
- 検索で矢印キーによるナビゲーションとEnterでタスクにジャンプ可能に

## [0.2.2] - 2025-01-29

### Added
- 10種類の新テーマを追加：Nord、Dracula、Monokai、Gruvbox、Tokyo Night、Catppuccin、Ocean、Sakura、MSX、PC-98
- TUI内でテーマ、モード、言語を切り替え可能（Sキー）

### Fixed
- MSXテーマが本物のTMS9918カラーパレットを使用するよう修正

### Changed
- Tursoデータベースインジケーターを視認性向上のため大文字に変更

## [0.2.1] - 2025-01-29

### Added
- Dキーでタスク削除（インライン確認ダイアログ付き）を追加
- タスク詳細ビューにステータスラベル表示を追加
- 国際化対応のためInfoタブの翻訳を追加

## [0.2.0] - 2025-01-29

### Added
- サブタスクへのナビゲーション付きプロジェクト詳細ビューを追加
- プロジェクト進捗バー（完了状況表示）を追加
- タスク検索機能（Ctrl+f）を追加
- 初回ユーザー向けセットアップウィザードを追加
- タスク詳細ビューからのプロジェクトリンクを追加
- ヘルプモーダルにWhat's Newタブ（変更履歴表示）を追加
- ビューモード切り替え用ModeSelectorコンポーネントを追加
- データベースリセットコマンド（`floq config reset-db`）を追加
- カンバンモードにタスク詳細ビューを追加
- GTDモードTUIにDoneタブを追加
- 連絡先入力付きMove-to-waiting機能を追加

### Changed
- タスク詳細フッターがテーマカラーを反映するよう変更
- What's Newタブにスクロールサポートを追加

### Fixed
- Tursoメタデータファイルの移行パスを修正

## [0.1.0] - 2025-01-29

### Added
- Floq（旧gtd-cli）の初回リリース
- GTDベースのタスク管理（Inbox/Next/Waiting/Someday/Doneワークフロー）
- カンバンボードビューモード
- 16種類以上のレトロテーマ（DOS、Commodore、Atariなど）
- 国際化サポート（英語・日本語）
- Tursoクラウド同期サポート
- サブタスク付きプロジェクト管理
- XDG Base Directory準拠
- タスクコメント機能
- 期限日サポート
